Get energy certificate data

Use this service to get data on:

  • Energy Performance Certificates (EPCs) for domestic properties
  • Commercial Energy Performance Certificates (CEPCs) for non-domestic properties
  • Display Energy Certificates (DECs) for public buildings

This data covers England and Wales. For data in Scotland, visit the Scottish EPC Register.

How you can use this service

You can use this service to:

  • download bulk certificate data which can be opened in Microsoft Excel
  • access certificate data using a developer API

If you need an energy certificate to sell or lease a property, visit the EPC Register.
