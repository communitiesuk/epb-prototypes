Get energy certificate data

Use this service to get data on:

  • Energy Performance Certificates (EPCs) for domestic and non-domestic properties
  • Display Energy Certificates (DECs) for public buildings

This data covers England, Wales and Northern Ireland

For properties in Scotland, use the Scottish energy certificate service.

How you can use this service

You can use this service to:

  • download certificate data as CSV files
  • access certificate data using a developer API

To check that a property complies with regulatory and legal requirements, visit the official register
